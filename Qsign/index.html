获取Sign

GetSign

可用版本: QQ9.0.50-QQ9.0.95
API 路由: https://api.52oicq.com/getSign
鉴权密钥: Authorization:鉴权密钥
请求方式: POST
请求体: JSON

请求参数

参数名值说明
uin11451411QQ号
qua114514QQqua
guid114514guid
cmd810_9命令代码
seq114514请求序列号
buffer114514待签名数据包
android_id114514Android设备唯一标识符

请求示例

{
  "uin": "11451411",
  "qua": "114514",
  "guid": "114514",
  "cmd": "810_9",
  "seq": "114514",
  "buffer": "114514",
  "android_id": "114514"
}

返回信息

返回参数

参数名值说明
code0响应状态码
msg""响应消息
data数据对象
├── token""访问令牌
├── extra""额外信息
├── sign""签名
└── o3did""第三方设备标识符

响应示例

{
    "code": 0,
    "msg": "",
    "data": {
        "token": "",
        "extra": "",
        "sign": "",
        "o3did": ""
    }
}

示例（Python）

from typing import Optional

class Qsign:
   import http.client
import json

conn = http.client.HTTPSConnection("api.52oicq.com")
payload = json.dumps({
   "uin": "11451411",
   "qua": "114514",
   "guid": "114514",
   "cmd": "810_9",
   "seq": "114514",
   "buffer": "114514",
   "android_id": "114514"
})
headers = {
   'Authorization': '鉴权密钥',
   'User-Agent': 'Apifox/1.0.0 (https://apifox.com)',
   'Content-Type': 'application/json'
}
conn.request("POST", "/getSign", payload, headers)
res = conn.getresponse()
data = res.read()
print(data.decode("utf-8"))

guid114514全局唯一标识符，用于标识应用或用户
android_id114514安卓设备ID，用于唯一标识一个安卓设备
